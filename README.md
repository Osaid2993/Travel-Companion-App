# Travel Companion App

## About the project
This is an Android Studio project created for Task 2.1P. The app allows users to convert values between different categories.

## Categories used
- Currency
- Fuel
- Liquid Volume
- Temperature

## Features
- Select a category
- Select from and to units
- Enter a value
- Press Convert to see the result

## Tools used
- Java
- XML

## How to run the app
1. Open the project in Android Studio
2. Let Gradle sync
3. Run the app on an emulator or Android device

## Conversion examples
- USD to AUD
- mpg to km/L
- Gallon (US) to Liters
- Celsius to Fahrenheit
